PostgreSQL Practical Task Summary

Module: Building Database Applications with PostgreSQL and Python

Overview
PostgreSQL was installed and configured successfully. Using SQL Shell (psql), several database operations were performed to understand database creation, table management, and data manipulation.

Tasks Performed

1. Created a database named 'demodb' using CREATE DATABASE.

2. Used PostgreSQL meta commands:
\l  - to list all available databases
\c  - to connect or switch to a database
\d  - to display tables inside the connected database

3. Created a table named 'students' with the following columns:
name (text)
number (integer)
age (integer)

4. Inserted records into the table using INSERT INTO command.

Example records:
Max | 10 | 20
May | 11 | 19
Sam | 12 | 20

5. Retrieved data using SELECT queries such as:
SELECT * FROM students;
SELECT name FROM students;
SELECT * FROM students WHERE age = 20;
SELECT number FROM students WHERE name = 'Max';

6. Practiced data deletion and cleanup using:
TRUNCATE TABLE students;
DROP DATABASE demodb;

Documentation
Screenshots were captured for each practical step showing the full SQL commands executed, the resulting output, and the system timestamp. All screenshots were taken without cropping or editing to meet submission requirements.

Conclusion
This practical exercise helped in understanding fundamental PostgreSQL operations including database creation, table creation, inserting data, retrieving data, and deleting records. These concepts form the basic foundation for building database-driven applications using PostgreSQL and Python.
