Combined Practical Summary
Module: Building Database Applications with PostgreSQL and Python

Overview
This practical work involved learning the basics of PostgreSQL database management and integrating it with Python using the psycopg2 library. The activities included installing required tools, performing database operations, and executing database queries both directly through PostgreSQL and through Python scripts in Visual Studio Code.

Activity 1: PostgreSQL Database Operations
PostgreSQL was installed and accessed through the SQL Shell (psql). The following operations were performed:

* Created a database using the CREATE DATABASE command.

* Used PostgreSQL meta commands such as:
  \l to list available databases
  \c to switch between databases
  \d to display tables within a database

* Created a table named students with fields such as name, number, and age.

* Inserted records into the table using INSERT INTO statements.

* Retrieved data using SELECT queries including filtering conditions.

* Removed data using TRUNCATE TABLE.

* Deleted the database using DROP DATABASE after completing the exercise.

Activity 2: Python Integration with PostgreSQL
Visual Studio Code was installed and used as the development environment. A Python virtual environment was created and the psycopg2 library was installed to enable communication between Python and PostgreSQL.

The following tasks were performed using Python scripts:

* Established a connection between Python and the PostgreSQL database.
* Created database tables using SQL queries executed through Python.
* Inserted records into the database using Python code.
* Retrieved and displayed data from the database.
* Collected user input through Python and inserted the data into the database.

Documentation
Screenshots were taken for all practical tasks showing the executed code, program output, and visible system timestamp. All screenshots were captured without cropping or editing as per the submission guidelines.

Conclusion
These activities helped in understanding basic database operations in PostgreSQL and demonstrated how Python can interact with databases to create, store, retrieve, and manage data for application development.
